jCVS Requirements

The requirements for running jCVS 5.0 are as follows:
  • Java 1.1, or Java 1.2

  • Swing 1.1 (built into Java 1.2)
    see http://www.gjt.org/req/swing/

  • Java Activation Foundation (JAF)
    see http://www.gjt.org/req/jaf/

  • JavaHelp
    see http://www.gjt.org/req/javahelp/
