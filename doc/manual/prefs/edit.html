Editting Preferences

Editting preferences in jCVS II is now simple and GUI based. You now use a preferences editor much like the one found in Netscape Navigator and other applications.

To edit the preferences, select the 'Edit Preferences...' menu item in the 'File' menu in the main window.

The preferences are organized into several categories:

  • Actions - preferences related to file actions, like double clicking to edit.

  • Advanced - preferences you should rarely need to configure.

  • Global - preferences that apply to global aspects of the application.

  • Project Window - preferences that control the project winow.

  • Work Bench - preferences that control the work bench panel in the main winow.
